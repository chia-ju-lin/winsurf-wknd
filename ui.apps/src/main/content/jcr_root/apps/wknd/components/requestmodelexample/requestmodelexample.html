Request-Based Sling Model Example

Properties from both the resource and request:

  • Title (from content): ${model.title || 'No title property found'}
  • Current Resource Path: ${model.currentResourcePath}

This component is using a Request-based Sling Model that can access properties from both the content repository and the HTTP request. To see request parameters in action, add ?param=something to the URL and refresh the page.
